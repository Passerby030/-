寄存器

{

RAM：“随机存储器”（掉电丢失，即时存储）类似于内存条

ROM：“只读存储器”（长期存储）类似于硬盘

}

**51单片机**

{

IO口，输入输出的口子，与引脚连接

P0，P1，P2，P3，每组八个引脚——（芯片内部引出与外部的接线）

中间是8051内核

}
